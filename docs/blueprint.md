# **App Name**: Eternal Union

## Core Features:

- Homepage Design: Replicate the homepage design from the provided link, including the layout and aesthetic.
- Inner Page Template: Build pages using the look-and-feel and layouts of the inner pages of the target URL.
- Preloader Animation: Implement a preloader animation with a white background and a heart design.
- Dynamic Content Population: Integrate CMS, so site content may be updated without modifying code.

## Style Guidelines:

- Primary color: Rose-pink (#F2A7B1), inspired by themes of love, weddings, and romance. This hue also evokes a traditional, vintage aesthetic.
- Background color: Light pink (#F8E8EB), subtly desaturated from the primary color, in keeping with a light color scheme.
- Accent color: Muted plum (#C68FB1), analogous to the primary rose-pink but with lower brightness and saturation, used to accent items like hover states.
- Font pairing: 'Playfair' (serif) for headlines, offering an elegant and fashionable touch, and 'PT Sans' (sans-serif) for body text, ensuring readability for longer text passages.
- Utilize elegant, heart-themed icons in a minimalistic style to reinforce the wedding theme.
- Employ a clean, grid-based layout, similar to the reference site, ensuring a balance of visual appeal and ease of navigation.
- Implement subtle, smooth animations for page transitions and interactive elements to enhance user engagement.
